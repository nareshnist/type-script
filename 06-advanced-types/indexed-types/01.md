# ⚠️ Important Point

```ts
const roles = {
  admin: "admin",
  user: "user"
};
```

👉 Idi **const variable**
❗ kani inside values change avvachu

```ts

```

---

# 🔥 Actual problem

👉 `const` means:

* variable change avvadu ❌
* **object values change avvachu ✅**

---

# 💡 Solution → `as const`

```ts
const roles = {
  admin: "admin",
  user: "user"
} as const;
```

---

## ✅ Ippudu emi jaruguthundi?

```ts

```

👉 ippudu:

* values lock ayyayi 🔒
* readonly ayyayi
* exact values preserve ayyayi
